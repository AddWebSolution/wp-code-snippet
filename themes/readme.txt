== Description ==
This addweb theme includes various sections such as a Hero section, Recent Blogs, Resource Archive, and Blog Signup.

== Installation ==
1. Download the theme zip file.
2. Upload the theme to your WordPress site.
3. Activate the theme.

== Sections Overview ==

1. **Hero Section:**
   - The Hero Section serves as the introductory section for the Resources page. It prominently displays a title, providing an engaging and visually appealing introduction to the content.

2. **Recent Blogs Section:**
   - The Recent Blogs Section dynamically showcases the most recent blog posts with featured images. Visitors can quickly access and explore the latest articles, fostering engagement and encouraging them to stay informed.

3. **Resource Archive Section:**
   - The Resource Archive Section offers a comprehensive collection of resources, categorized based on topics, content types, industries, and personas. Users can explore and filter resources using a search form, enhancing accessibility and facilitating targeted content discovery.

4. **Blog Signup Section:**
   - The Blog Signup Section provides users with an opportunity to stay connected with Digibee by signing up for updates. It integrates HubSpot forms for a seamless user registration experience, ensuring that visitors can receive the latest news and insights from Digibee.

== Usage ==
- Customize theme settings in the WordPress Customizer to tailor the appearance and behavior of the theme according to specific preferences.
- Adjust content and styling in theme files as needed to align with the branding and content strategy.
- Ensure all necessary plugins are installed and activated to leverage the full functionality of the theme.
